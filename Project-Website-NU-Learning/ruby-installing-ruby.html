Installing Ruby

Introduction

Before diving into Ruby programming, the first essential step is installing Ruby correctly on your system. This guide will walk you through the process of setting up Ruby on different operating systems, ensuring you’re ready to write and execute your Ruby code with ease. Whether you’re on Windows, macOS, or Linux, we’ve got you covered!

We'll cover three main installation methods:

  • Using a version manager (recommended for flexibility)
  • Installing Ruby directly from the operating system's package manager
  • Manual installation (for customized configurations)

Installing on macOS

Step 1: Check if Ruby is already installed

macOS comes with Ruby preinstalled, but it’s usually not the latest version. You can check your Ruby version by opening the terminal and typing:

$ ruby -v

If Ruby is installed, you’ll see a version number. If it’s outdated or not installed, continue with the next steps.

Step 2: Install Homebrew

Homebrew is a package manager for macOS that makes it easy to install software. If you don’t have it yet, install it by running this command in your terminal:

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 3: Install Ruby using Homebrew

With Homebrew installed, you can now install the latest version of Ruby by running:

$ brew install ruby

Once the installation is complete, verify the installation by checking the Ruby version again:

$ ruby -v

You should now have the latest version of Ruby installed on your system.

Installing on Windows

Windows users can install Ruby via RubyInstaller, which provides a simple way to get Ruby running on Windows.

Step 1: Download RubyInstaller

Visit the official RubyInstaller website at rubyinstaller.org and download the recommended version.

Step 2: Run the Installer

Open the downloaded file and follow the installation instructions. Make sure to check the box that says "Add Ruby executables to your PATH."

Step 3: Verify the Installation

Once installed, open the command prompt and verify Ruby is installed by running:

ruby -v

This should display the version number, confirming Ruby is installed on your Windows machine.

Installing on Linux

Linux users can install Ruby via their distribution’s package manager. The following guide provides instructions for Ubuntu.

Step 1: Update Your Package List

Before installing Ruby, make sure your package list is up-to-date by running:

$ sudo apt update

Step 2: Install Ruby

Now, install Ruby by running:

$ sudo apt install ruby-full

Step 3: Verify the Installation

After installation, check your Ruby version to ensure everything is set up correctly:

$ ruby -v

Using RVM for Managing Versions

RVM (Ruby Version Manager) is a popular tool for managing different versions of Ruby on the same system.

Step 1: Install RVM

To install RVM, first install the required dependencies:

$ sudo apt-get install curl gpg

Then run this command to install RVM:

$ \curl -sSL https://get.rvm.io | bash -s stable

Step 2: Install Ruby via RVM

With RVM installed, you can install the latest version of Ruby by running:

$ rvm install ruby

Step 3: Set Ruby Version

To set the default Ruby version for your system, use:

$ rvm use ruby --default

RVM allows you to easily switch between different Ruby versions, making it a flexible option for developers working on multiple projects.

Conclusion

Congratulations! You’ve successfully set up Ruby on your system. Whether you're using macOS, Windows, or Linux, you’re now ready to start writing Ruby code. Remember, proper installation is the foundation for everything you’ll build in Ruby, so double-check that everything is working correctly before moving forward.

In the next lesson, we’ll dive into running your first Ruby scripts, and from there, you’ll quickly see the power and simplicity of Ruby in action. Let’s get coding!
