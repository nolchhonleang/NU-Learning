Object Basics

Foundations Course

Introduction

Welcome to NU-Learning's in-depth lesson on Object Basics in JavaScript! This lesson will introduce you to one of the fundamental concepts in programming: Objects. Objects are the building blocks of JavaScript, enabling you to represent real-world entities in code. Mastering objects is crucial as they form the basis for many features in web development, such as creating and managing dynamic data structures, encapsulating functionalities, and interacting with APIs.

By the end of this lesson, you will understand what objects are, how to create and use them, and the many ways they can enhance your JavaScript skills. Let's begin our journey into the world of Object Basics!

What is an Object?

An object in JavaScript is a collection of related data and functions, organized in key-value pairs. In simpler terms, an object can hold both properties (which are variables) and methods (which are functions), allowing you to group related information and behavior together.

For example, imagine you want to represent a "car" in code. A car has properties like 'color', 'brand', and 'year'. It also has actions or methods, like 'start' or 'drive'. In JavaScript, we can encapsulate all this information into an object:


let car = {
    color: 'red',
    brand: 'Toyota',
    year: 2020,
    start: function() {
        console.log('The car is starting...');
    },
    drive: function() {
        console.log('The car is driving...');
    }
};

In the example above, 'color', 'brand', and 'year' are properties of the car object, while 'start' and 'drive' are methods. Objects allow us to organize and structure our code in a more intuitive way, mimicking real-life objects and their behaviors.

Creating Objects

There are several ways to create objects in JavaScript. The most common way is using object literals, as shown in the previous example. However, JavaScript also provides other methods to create objects, such as using the new Object() syntax or object constructors.

Using Object Literals

The object literal syntax is the easiest and most straightforward way to create an object:


let person = {
    firstName: 'Kimseng',
    lastName: 'Ear',
    age: 20
};

This creates a person object with properties firstName, lastName, and age.

Using the new Object() Constructor

Another way to create an object is by using the new Object() syntax, although it's less commonly used:


let animal = new Object();
animal.species = 'Dog';
animal.name = 'Buddy';
animal.age = 3;

This creates an animal object and assigns its properties afterward.

Accessing Object Properties

Once you have created an object, you can access its properties using either dot notation or bracket notation.

Dot Notation

Dot notation is the most common way to access object properties:


console.log(person.firstName);  // Output: Kimseng

In this case, we accessed the firstName property of the person object.

Bracket Notation

Bracket notation is another way to access object properties, often useful when the property name is dynamic:


console.log(person['lastName']);  // Output: Ear

Bracket notation is especially useful if the property name contains special characters or spaces.

Modifying Object Properties

Once you have created an object, you can modify its properties by simply reassigning new values:


person.age = 20;
console.log(person.age);  // Output: 20

In this example, we changed the age property of the person object.

Methods in Objects

Methods are functions that are stored as properties of an object. They allow objects to have behaviors:


let calculator = {
    add: function(a, b) {
        return a + b;
    },
    subtract: function(a, b) {
        return a - b;
    }
};

console.log(calculator.add(5, 3));  // Output: 8
console.log(calculator.subtract(9, 4));  // Output: 5

Here, the calculator object contains two methods, add and subtract, that perform arithmetic operations.

Iterating Over Objects

You can loop through an object’s properties using the for...in loop, which is specifically designed for iterating over object keys:


let book = {
    title: 'JavaScript Basics',
    author: 'Kimseng Ear',
    year: 2024
};

for (let key in book) {
    console.log(key + ': ' + book[key]);
}

This will output the key-value pairs of the book object:


title: JavaScript Basics
author: Kimseng Ear
year: 2024

Conclusion

In this lesson, we explored the basics of objects in JavaScript. You learned how to create objects, access and modify their properties, define methods, and iterate over their keys. Objects are a powerful and flexible way to organize data in your JavaScript programs. With this foundational knowledge, you are well-prepared to tackle more advanced topics such as prototypes, classes, and object-oriented programming.
